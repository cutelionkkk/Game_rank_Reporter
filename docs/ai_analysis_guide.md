# AI Agent 分析指南

## 概述

Game Rank Reporter 抓取的数据可以导出为结构化 JSON，供 AI Agent 做深度分析。这比纯排名变动报告更有价值——AI 可以发现人类难以察觉的趋势和关联。

## 使用方式

```bash
# 1. 先抓取最新数据
python run.py --crawl

# 2. 导出分析数据
python run.py --export-analysis

# 3. 数据保存在 analysis_data.json
```

然后把 `analysis_data.json` 的内容喂给你的 AI Agent，配合下面的分析 prompt 使用。

## 数据结构

`analysis_data.json` 包含：

```json
{
  "metadata": {
    "crawl_time": "2026-02-27T06:36:54Z",
    "previous_crawl_time": "2026-02-26T18:36:54Z",
    "country": "us",
    "top_n": 100
  },
  "charts": {
    "ios_free": { "current": [...], "changes": {...} },
    "ios_paid": { "current": [...], "changes": {...} },
    "ios_grossing": { "current": [...], "changes": {...} },
    "gp_free": { "current": [...], "changes": {...} },
    "gp_paid": { "current": [...], "changes": {...} },
    "gp_grossing": { "current": [...], "changes": {...} }
  },
  "category_trends": { ... },
  "consecutive_risers": { ... }
}
```

每个 app 条目包含：
- `rank` — 当前排名
- `app_name` — 游戏名称
- `developer` — 开发商
- `category` — 品类（如 Strategy, Puzzle, Casual）
- `rating` — 评分
- `rating_count` — 评分数量
- `price` — 价格
- `prev_rank` — 上次排名（如有）
- `rank_change` — 排名变化（正数=上升）

## 分析 Prompt 模板

### 模板 1：市场洞察报告

```
你是一个游戏市场分析师。以下是美国区 iOS App Store 和 Google Play 游戏排行榜的最新数据。

请分析以下内容，并以简洁的中文报告输出：

1. **市场热点**
   - 本次抓取中最引人注目的变化是什么？
   - 有哪些游戏的排名变化值得深挖？为什么？

2. **品类分析**
   - 哪些游戏品类正在上升？哪些在衰退？
   - 有没有新兴品类出现的信号？
   - 各品类在免费榜 vs 畅销榜的表现差异说明什么？

3. **开发商分析**
   - 哪些开发商有多款游戏在榜？说明什么？
   - 有没有小开发商的游戏异军突起？

4. **跨平台比较**
   - iOS 和 GP 的榜单有哪些显著差异？
   - 有没有游戏在一个平台表现好但另一个不好？原因可能是什么？

5. **市场机会**
   - 基于当前数据，你认为哪些品类/玩法方向有机会？
   - 如果要做一款新游戏，你会建议什么方向？

数据如下：
{analysis_data.json 的内容}
```

### 模板 2：竞品追踪

```
以下是最新的游戏排行榜数据。我在关注以下竞品：
- [竞品1名称]
- [竞品2名称]
- [竞品3名称]

请分析：
1. 这些竞品当前的排名位置和变化趋势
2. 它们在免费榜/付费榜/畅销榜的表现差异
3. 同品类内的竞争格局
4. 近期排名变化可能的原因（新版本？营销活动？行业事件？）
5. 对我方产品的建议

数据如下：
{analysis_data.json 的内容}
```

### 模板 3：投资视角

```
你是一个游戏行业投资分析师。以下是美国区游戏排行榜数据。

请从投资角度分析：
1. **头部公司表现** — 畅销榜 Top 20 里哪些开发商/发行商占主导？
2. **新晋玩家** — 有没有新公司或新 IP 进入头部？
3. **品类生命周期** — 哪些品类处于增长期、成熟期、衰退期？
4. **变现能力** — 免费榜排名高但畅销榜排名低的游戏，可能说明什么？反过来呢？
5. **风险信号** — 有没有曾经的头部游戏在快速下滑？

数据如下：
{analysis_data.json 的内容}
```

## 自动化分析流程

如果你的 AI Agent 支持定时任务（如 OpenClaw cron），可以设置完全自动化的流程：

```
1. 抓取数据: python run.py --crawl
2. 导出分析数据: python run.py --export-analysis
3. 读取 analysis_data.json
4. 使用上述 prompt 模板生成分析报告
5. 把分析报告推送到用户指定的渠道
```

这样你不仅会收到排名变动通知，还会收到一份 AI 撰写的深度分析报告。

## 分析方法论

### 识别市场机会的 5 个维度

1. **品类空白** — 畅销榜有收入但免费榜同品类少 → 用户有付费意愿但选择不多
2. **上升趋势** — 某品类连续多次增加在榜游戏数量 → 市场在扩张
3. **跨平台差异** — iOS 有但 GP 没有（或反过来）→ 一个平台的成功经验可能迁移
4. **新游戏验证** — 上线 30 天内进入 Top 100 → 说明该品类/玩法有市场需求
5. **开发商集中度** — 某品类被少数大厂垄断 vs 多个小团队 → 竞争壁垒不同

### 排名变化的常见原因

| 变化模式 | 可能原因 |
|---------|---------|
| 突然飙升 50+ 名 | 新版本/大更新、营销活动、被推荐、病毒传播 |
| 缓慢持续上升 | 口碑传播、自然增长、SEO/ASO 优化生效 |
| 突然暴跌 | 负面事件、服务器问题、政策限制、竞品上线 |
| 缓慢下滑 | 内容老化、用户流失、竞品蚕食 |
| 畅销榜高但免费榜低 | 硬核用户群但新用户获取困难 |
| 免费榜高但畅销榜低 | 用户量大但变现效率低 |
